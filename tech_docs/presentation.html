The Vision Pipeline

How SortSmartSE sees and understands waste in real-time.

1. Dual-Pass Analysis

We combine Object Detection (spatial awareness) with Image Labeling (classification) for maximum precision.

2. Smart Filtering

Advanced exclusion logic filters out "noise" like furniture, pets, or hands, focusing only on Waste-related artifacts.

3. Detection Boost

Confidence scores are dynamically boosted by 10% if a prominent object is detected in the viewfinder frame.
